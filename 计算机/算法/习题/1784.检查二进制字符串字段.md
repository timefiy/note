# 检查二进制字符串字段

> [!question] 题目描述
> 给你一个二进制字符串 s ，该字符串不含前导零 。
>
> 如果 s 包含 零个或一个由连续的 '1' 组成的字段 ，返回 true 。否则，返回 false 。
>
> 如果 s 中 由连续若干个 '1' 组成的字段 数量不超过 1 ，返回 true。否则，返回 false。
>
>---
> [!light] 核心思路
> **破局点**：遍历字符串，检查是否出现多个由连续 '1' 组成的字段。
>
> 1. 使用状态标记：0-未开始，1-在1字段中，2-已结束1字段
> 2. 遍历字符串，根据当前字符和状态更新状态
> 3. 如果在状态2时遇到 '1'，说明有多个1字段，返回 false

> [!code] 代码实现
>
> ```cpp
> class Solution {
> public:
>     bool checkOnesSegment(string s) {
>         int flag = 0;
>         for(char c : s){
>             if(c == '1' && flag == 0) flag = 1;
>             else if(flag ==1 && c == '0') flag = 2;
>             else if(flag == 2 && c == '1') return false;
>         }
>         return true;
>     }
> };
> ```
