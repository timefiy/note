# 找出所有稳定的二进制数组 I

> [!question] 题目描述
> ![](assets/3129.找出所有稳定的二进制数组-i/file-20260311210507514.png)
>

---

> [!light] 核心思路
> **破局点**：使用记忆化搜索，考虑数组末尾是0还是1，并枚举连续相同数字的长度
>
> 1. 定义dp(zero, one, end)表示用zero个0和one个1构建数组，末尾为end的稳定数组数量
> 2. 递归时枚举末尾连续相同数字的长度k（1到limit）
> 3. 如果末尾是0，则前一个必须是1，状态转移为dp(zero-k, one, 1)
> 4. 如果末尾是1，则前一个必须是0，状态转移为dp(zero, one-k, 0)
> 5. 使用三维数组memo进行记忆化，避免重复计算
> 6. 边界条件：当某个数字用完时，检查另一个数字是否不超过limit

---

> [!code] 代码实现

```cpp
> # include <bits/stdc++.h>
> using namespace std;
> 
> /*
>  * @lc app=leetcode.cn id=3129 lang=cpp
>  *
>  * [3129] 找出所有稳定的二进制数组 I
>  */
> 
> // @lc code=start
> class Solution {
>     int memo[205][205][2];
>     int MOD = 1e9 + 7;
> 
> public:
>     int numberOfStableArrays(int zero, int one, int limit) {
>         memset(memo, -1, sizeof(memo));
>         
>         return (dp(zero, one, 0, limit) + dp(zero, one, 1, limit)) % MOD;
>     }
> 
>     int dp(int zero, int one, int end, int& limit) {
>         if (zero == 0) return (end == 1 && one <= limit) ? 1 : 0;
>         if (one == 0) return (end == 0 && zero <= limit) ? 1 : 0;
> 
>         if (memo[zero][one][end] != -1) {
>             return memo[zero][one][end];
>         }
> 
>         long long res = 0;
>         
>         if (end == 0) {
>             for (int k = 1; k <= min(zero, limit); ++k) {
>                 res = (res + dp(zero - k, one, 1, limit)) % MOD;
>             }
>         } else {
>             for (int k = 1; k <= min(one, limit); ++k) {
>                 res = (res + dp(zero, one - k, 0, limit)) % MOD;
>             }
>         }
> 
>         return memo[zero][one][end] = res;
>     }
> };
// @lc code_end
```
