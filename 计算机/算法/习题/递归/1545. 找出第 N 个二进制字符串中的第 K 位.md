# 找出第 N 个二进制字符串中的第 K 位

> [!question] 题目描述
> ![](assets/1545.%20找出第%20N%20个二进制字符串中的第%20K%20位/file-20260303222819417.png)
>
>---

<br>

> [!light] 核心思路
> **破局点**：使用 **递归构造** 按照规则生成字符串序列。
>
> 1. 理解递归构造规则：Si = Si-1 + "1" + reverse(invert(Si-1))
> 2. 从 S1 = "0" 开始，逐步构造到 Sn
> 3. 实现字符串反转和位翻转操作
> 4. 返回第 k 位字符（注意索引从 1 开始）

<br>

> [!code] 代码实现
> ```cpp
> class Solution {
> public:
>     char findKthBit(int n, int k) {
>         string s = "0";
>         s = findKthString(n, s);
>         return s[k - 1];
>     }
> 
>     string findKthString(int n, string &s){
>         if (n == 0) {
>             return s;
>         }
> 
>         string temp = s;
>         string inverted = invert(temp);
>         reverse(inverted.begin(), inverted.end());
>         s = s + "1" + inverted;
> 
>         return findKthString(n - 1, s);
>     }
> 
>     string invert(string s) {
>         for (int i = 0; i < s.size(); i++) {
>             if (s[i] == '0') {
>                 s[i] = '1';
>             } else {
>                 s[i] = '0';
>             }
>         }
>         return s;
>     }
> };
> ```
