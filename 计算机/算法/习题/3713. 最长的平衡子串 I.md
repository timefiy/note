# 3713. 最长的平衡子串 I

## 题目

![题目图片](assets/3713.%20最长的平衡子串%20I/file-20260212233619116.png)

## 解答

```cpp
class Solution {
public:
    int longestBalanced(string s) {
        int n = s.size();
        int ans = 0;

        for(int i = 0; i < n; i++){
            int cnt[26] = {0}; 
            
            for (int j = i; j < n; j++)
            {
                cnt[s[j] - 'a']++;

                if(check(cnt)){
                    ans = max(ans, j - i + 1);
                }
            }
        } 
        return ans;
    }

    bool check(int cnt[]){
        int standard = -1; 
        for (int i = 0; i < 26; i++)
        {
            if (cnt[i] > 0) {
                if (standard == -1) {
                    standard = cnt[i]; 
                } else if (cnt[i] != standard) {
                    return false; 
                }
            }
        }
        return standard != -1;
    }
};
```
