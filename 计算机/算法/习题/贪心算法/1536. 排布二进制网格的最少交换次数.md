# 排布二进制网格的最少交换次数

> [!question] 题目描述与截图
>
> ![](assets/1536.%20排布二进制网格的最少交换次数/file-20260302195115045.png)
>
> ---

<br>

> [!light] 核心思路
> **破局点**：使用 **贪心策略** 逐行处理，确保每一行都满足主对角线上方全为 0 的条件。
>
> 1. 从第 0 行开始，逐行检查是否满足条件（第 i 行从第 i+1 列开始必须全为 0）
> 2. 如果当前行不满足条件，在下方寻找第一个满足条件的行
> 3. 通过相邻行交换将目标行移动到当前位置
> 4. 累计交换次数，如果找不到合适的行则返回 -1

<br>

>[!example] 代码实现
>
> ```cpp
> class Solution {
> public:
>     int minSwaps(vector<vector<int>>& grid) {
>         int n = grid.size();
>         int swaps = 0;
>         
>         for(int i = 0; i < n; i++){
>             if(!checkrow(grid[i], i)){
>                 int j = findrow(grid, i);
>                 if(j == -1) return -1;
>                 
>                 // 将第j行逐步交换到第i行
>                 for(int k = j; k > i; k--){
>                     swap(grid[k], grid[k-1]);
>                     swaps++;
>                 }
>             }
>         }
>         return swaps;
>     }
> 
>     bool checkrow(vector<int>& row, int n){
>         for (int i = n + 1; i < row.size(); i++){
>             if(row[i] == 1) return false;
>         }
>         return true;
>     }
> 
>     int findrow(vector<vector<int>>& grid, int n){
>         for (int i = n + 1; i < grid.size(); i++){
>             if(checkrow(grid[i], n)){
>                 return i; 
>             }
>         }
>         return -1;
>     }
> };
> ```
