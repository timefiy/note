# 1536. 排布二进制网格的最少交换次数

![](assets/1536.%20排布二进制网格的最少交换次数/file-20260302195115045.png)

## 题目分析

给定 n×n 二进制网格，通过相邻行交换使主对角线上方全为 0。

**关键点：**
- 只能交换相邻行
- 目标：主对角线上方全为 0
- 第 i 行从第 i+1 列开始必须全为 0

## 解题思路

**贪心策略：**
1. 从第 0 行开始，逐行检查
2. 如果当前行不满足条件，找到下方第一个满足条件的行
3. 通过相邻交换将其移到当前位置
4. 累计交换次数

## 代码实现

```cpp
class Solution {
public:
    int minSwaps(vector<vector<int>>& grid) {
        int n = grid.size();
        int swaps = 0;
        
        for(int i = 0; i < n; i++){
            if(!checkrow(grid[i], i)){
                int j = findrow(grid, i);
                if(j == -1) return -1;
                
                // 将第j行逐步交换到第i行
                for(int k = j; k > i; k--){
                    swap(grid[k], grid[k-1]);
                    swaps++;
                }
            }
        }
        return swaps;
    }

    bool checkrow(vector<int>& row, int n){
        for (int i = n + 1; i < row.size(); i++){
            if(row[i] == 1) return false;
        }
        return true;
    }

    int findrow(vector<vector<int>>& grid, int n){
        for (int i = n + 1; i < grid.size(); i++){
            if(checkrow(grid[i], n)){
                return i; 
            }
        }
        return -1;
    }
};
```

## 复杂度分析

- **时间复杂度：** O(n³)
  - 外层循环 O(n)
  - 查找合适行 O(n)
  - 相邻交换 O(n)
- **空间复杂度：** O(1)
