# 二进制矩阵中的特殊位置

> [!question] 题目描述
> 给你一个大小为 `rows x cols` 的矩阵 `mat`，其中 `mat[i][j]` 是 `0` 或 `1`，请返回矩阵 `mat` 中特殊位置的数目。
>
> 特殊位置定义：如果 `mat[i][j] == 1` 并且第 `i` 行和第 `j` 列中的所有其他元素均为 `0`（行和列的下标均从 `0` 开始），则位置 `(i, j)` 被称为特殊位置。
>
>---

<br>

> [!light] 核心思路
> **破局点**：使用 **暴力枚举** 检查每个位置是否为特殊位置。
>
> 1. 遍历矩阵中的每个位置
> 2. 对于值为 1 的位置，检查其所在行和列是否只有这一个 1
> 3. 如果满足条件，计数器加 1

<br>

> [!code] 代码实现
> ```cpp
> class Solution {
> public:
>     int numSpecial(vector<vector<int>>& mat) {
>         int ans = 0;
>         for (int i = 0; i < mat.size(); i++)
>         {
>             for (int j = 0; j < mat[0].size(); j++)
>             {
>                 if (mat[i][j] == 1){
>                     if (check(mat, i, j))
>                         ans++;
>                 }
>             }
>         }
>         return ans;
>     }
> 
>     bool check(vector<vector<int>>& mat, int x, int y){
>         for (int i = 0; i < mat.size(); i++)
>         {
>             if (i != x && mat[i][y] == 1)
>                 return false;
>         }
>         for (int j = 0; j < mat[0].size(); j++)
>         {
>             if (j != y && mat[x][j] == 1)
>                 return false;
>         }
>         return true;
>     }
> };
> ```
