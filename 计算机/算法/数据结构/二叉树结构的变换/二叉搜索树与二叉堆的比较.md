# 二叉搜索树与二叉堆的比较

| **特性**   | **二叉搜索树 (BST)**               | **二叉堆 (Binary Heap)**             |
| -------- | ----------------------------- | --------------------------------- |
| **排序规则** | **左右关系**：左子节点 < 根 < 右子节点      | **上下关系**：根节点必须是最大或最小（父 > 子或父 < 子） |
| **主要用途** | 动态查找、排序、范围查询                  | 优先级队列、堆排序、求前 K 个最值                |
| **结构限制** | 形状不固定（可能退化成链表），除非是平衡 BST      | **必须是完全二叉树**（除了最后一层外，其余层全满）       |
| **查找效率** | 平均 $O(\log n)$，最差 $O(n)$      | 查找特定元素效率低 $O(n)$                  |
| **最值访问** | 查找最小/最大需 $O(\log n)$ 或 $O(h)$ | 访问最值（根节点）仅需 $O(1)$                |
