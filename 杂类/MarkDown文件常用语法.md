# Markdown 常用语法

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 文本格式

```markdown
*斜体*
**粗体**
***粗斜体***
~~删除线~~
==高亮==
`行内代码`
```

*斜体*
**粗体**
***粗斜体***
~~删除线~~
==高亮==
`行内代码`

## 列表

```markdown
- 无序列表项1
- 无序列表项2
  - 嵌套项

1. 有序列表项1
2. 有序列表项2
   1. 嵌套项
```

## 链接和图片

```markdown
[链接文本](https://example.com)
![图片描述](image.jpg)
```

## 代码块

````markdown
```语言名
代码内容
```
````

## 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
```

## 引用

```markdown
> 这是引用内容
> 
> 可以多行
```

## 分割线

```markdown
---
```

## 任务列表

```markdown
- [x] 已完成任务
- [ ] 未完成任务
```

## 转义字符

```markdown
\* \_ \# \` \[ \]
```
